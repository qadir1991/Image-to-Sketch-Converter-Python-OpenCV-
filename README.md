🎨 Image to Sketch Converter (Python + OpenCV)

📌 Description


This is a Python-based image processing application that allows users to upload an image and convert it into a pencil sketch.

The program uses a file picker to let the user select an image, processes it using OpenCV, and generates a sketch-style output.

This project is ideal for beginners to learn basic image processing techniques.

🚀 Features

📂 Upload image using file picker (Tkinter)

🖼️ Convert image to pencil sketch

💾 Automatically saves output image

👀 Displays original and sketch images

🧠 Simple and clean logic

🛠️ Requirements

Python 3.x

OpenCV

Install dependency:

pip install opencv-python

▶️ How to Run

Clone the repository:

git clone https://github.com/your-username/image-to-sketch.git

Go to project folder:

cd image-to-sketch

Run the program:

python sketch.py

🎯 How It Works

A file picker window opens

User selects an image

Program processes the image:

Convert to grayscale

Invert image

Apply blur

Generate sketch

Final sketch is displayed and saved
